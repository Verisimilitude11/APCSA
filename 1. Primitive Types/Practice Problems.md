# Practice Problems for Unit 1: Primitive Types

## Problem 1

Which of the following are not valid declarations for variables of primitive types in Java?

(A) `double a = 3.14;`

(B) `float b = 2.718;`

(C) `int c = "hello";`

(D) `char d = 'd';`

(E) `long e = 1234567890L;`

Answer: (C)
